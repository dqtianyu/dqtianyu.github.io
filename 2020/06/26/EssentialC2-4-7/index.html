Essential C++ 第二章C2.4-2.7

局部静态对象

不想在函数中间来回通信，而去在file scope中定义，永远是一种冒险。
局部静态变量，在局部定义前，加上static,使它的值调用结束时候，不会消失，这里的变量与全局变量 共享一个数据区。

Inline函数

&nbsp；&nbsp；把一个大函数拆分成一个个小函数，为了减少对函数间来回调用影响的速度，在函数前面加上inline,函数在被调用时候会展开（这样会占用更多内存空间，或者指令缓存），函数经常调用时候，可以尝试这样的操作。
使用 inline 后函数调用速度会加快 。
inline只是一种请求，得看编译器愿不愿意。

重载函数

参数列表不相同的函数，可以拥有同样的函数名称，具体调用的时候，让编译器去考虑。
重载是对不同的参数列表做不同的处理，如果仅仅是改变其中的数据类型，主体不变，用下面的function template

定义使用模板函数

function template 可以将参数列表中指定的部分（或全部）参数类型抽出来。
下例中我们抽出来vector所持的元素类型，这个类型由用户提供。
template 后的< >可以跟一个或多个标识符。 

template <typename elemType>
void display_message(const string &msg, const vector<elemType> &vec)
{
    cout << msg;
    for (int ix = 0; ix < vec.size(); ++ix)
    {
        elemType t = vec[ix];
        cout << t <<' ';
    }
}

关键字typename表示elemType在display_message()函数中，是一个暂时放置 类型 的占位符，elemType只是个任意名称。
如何使用这种函数?
正常用就行，编译器会分析用户输入的vector类型，如下：

int main()
{
    string imsg = "hello\n";
    vector<int> ivec;
    display_message(imsg,ivec);

    system("pause");
}
