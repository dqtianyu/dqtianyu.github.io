利用功能连接对脑疾病进行分类和预测


本文的内容：

  • 进行了对FC预测和分类研究的全面综述，以对现有的大脑功能的处理方法和典型的分类策略进行指导。
  • 阐述了目前最先进的功能连接分析方法：
    • 静态功能连接(static functional connectivity，SFC)
    • 动态功能连接(DFC)

研究背景：
脑功能成像技术，已成为研究脑疾病的重要工具：

  • 功能磁共振成像(fMRI)
  • 正电子发射断层扫描(PET)
  • 脑电图(EEG)

  我们非常希望利用功能性神经成像数据，揭示的大脑功能连接，用来表征大脑功能异常，进而有利于诊断和治疗。
  其中fMRI能够以高空间分辨率对大脑功能进行非侵入性研究，并已广泛用于检测和描述大脑网络或功能关联区域之间的连接

如何从大量的测量数据中选择特征，作为生物标志物来建立模型，对脑功能障碍进行分类或预测是一个重要而具有挑战性的问题。

  • 分类：对在扫描时或扫描前已经获得的疾病标签进行分类
  • 预测：是预测未知的疾病标签、未来进展或连续变化

与分类相比，预测更困难，但却是更有希望的临床应用。

本文是对脑功能连接的各种测量方法和典型的分类策略进行综述
   通常情况下，功能连接被认为在扫描时间(通常是几分钟)内是固定的，而之前的大多数fMRI研究都采用了静态功能连接(SFC)分析。直到最近，越来越多令人兴奋的研究已经证明，将大脑功能连接视为随时间变化的动态，可以成功地揭示正常人类大脑在疾病状态下受到的干扰。

  • 一些名词：
    • 感兴趣区（ROI）： area of interest
        指研究者感兴趣的地方。
    • 功能连接（FC）： functional connection
        功能链接就是把某个脑区做成ROI，也叫种子点(seed region)，提取出其信号随时间点变化的序列(timecourse)，然后用这个序列去和全脑体素做相关。
    • 静态功能连接（SFC）：static functional connectivity
    • 动态功能连接（DFC）：dynamic functional connection
    • 体素（Voxel）：数字数据于三维空间分割上的最小单位
    • 主成分分析（PCA）：Principal components analysis

1. 静态功能连接(SFC)

1.1 模型驱动的种子点间的功能连接分析

  模型驱动法是用先验知识来确定一个大脑区域或体素的集合区作为ROI，然后探究大脑其他脑区与ROI功能连接。

  • 三个关键步骤：
    1. 确定ROI位置和形状或voxel的位置
    2. 计算ROI的时间序列
    3. 评估不同ROI之间的功能连接或ROI与大脑内的所有其他体素或脑区的功能连接

  上述步骤结束后，利用种子点法选择特定的ROI，计算与其他大脑脑区功能连接变化，所得功能连接强度可反映所选择的ROI及与其他大脑脑区之间的时间变动关系，这可作为分类和预测大脑精神疾病的特征。

  • 使用种子点法，可以用PCA在一定程度上降低ROI的代表性时间序列中的噪声效应
  • 遇到的挑战：如何确定一个具有一致脑功能体素的合理脑区
  • 两个代表性时间序列之间的功能连接主要是通过计算相关性来估计其线性关系，也可以通过互信息方法来评估其非线性关系。

1.2 数据驱动的基于 Spatial Functional Network Maps 的分析

和模型驱动的区别：数据驱动的方法估计功能网络不需要预先定义大脑区域或体素。

流行的方法：

  • 空间独立成分分析(ICA)
  • 主成分分析(PCA)
  • 和聚类方法

相对于ROI-based的方法， ICA方法的优点包括：

  1. 不需要先验知识
  2. 从整个大脑同时估计多个网络数据
  3. 重叠的成分,可以提供一个空间滤波的基础
  4. 被试之间成分的自适应性，允许捕获区域内的被试间可变性。

ICA的缺点：ICA以任意顺序生成ICs
解决方案：

  1. 第一种策略是对每个被试分别执行ICA，然后使用主观识别等方法建立各被试之间的ICs对应关系
  2. 组ICA（GIG-ICA）

1.3功能网络连接（FNC）分析

  功能网络连接(FNC)分析采用了一种结合模型驱动和数据驱动方法的策略。
主要步骤：

  1. 首先对受试者的fMRI数据进行组ICA分析得到受试者特异性功能网络(由ICs反映)及其相关波动[由时间序列(time courses，TCs)反映；
  2. 通过计算其后处理的TCs之间的Pearson相关性的连接性度量来获得任意两个网络之间的功能连接，从而产生包括所有网络之间的连接性强度的功能连接矩阵，FNC与ICA都需要预先确定组件的数量。

FNC与ROI区别：

  • FNC分析中，将数据驱动的方法应用于fMRI数据，生成功能共激活的大脑区域
  • 在基于ROI的方法中，大脑区域通常是通过先验知识来决定的

1.4其他的功能连接测量方法

例如：区域一致性(ReHo)被提议用来反映区域功能连接性(或协同激活)

2. 动态功能连接(DFC)

计算时间分辨率下的大脑不同脑区的连通性度量。

2.1 基于滑动窗方法的动态功能连接分析

基本流程：

  1. 通过测量滑动窗口中ROI或体素之间的功能连接来评估DFC，从而产生多个连接矩阵，
  2. 对不同窗口中的fMRI数据执行ICA(或IVA)以生成动态空间网络模式，或对网络的时间序列(即时间序列)进行分段。
  3. 计算时变的FNC

提取连接状态的方法：

  • 聚类
    • 缺点：
      • 当处理不一定具有理想分布的有噪声数据时，聚类方法可能无法收敛。
      • 聚类分析的一个更严重的缺点是，该方法总是可以产生具有任意数量集群的分区，而不管使用的特性是否显示指示聚类的模式。
  • 主成分分析(PCA)
  • Fisher discrimination dictionary learning (FDDL)
  • 空间和时间独立成分分析(ICA)

此外一些研究表明：低频震幅(ALFF)和FNC随时间的变化存在相关性。

2.2 用于提取动态连接性的无窗口方法

  • 方法提出原因：滑动时间窗方法中，窗口太长太短都不行。

  • 无窗口的方法其概念是将观测窗口与原始时间过程的频率相适应。

  • 主要优点：可获取任意变化率的功能连接。无窗口操作可以解决缺乏窗口长度标准的问
    题。

3. 分类和预测

3.1特征选择策略

3.1.1 Filter方法（筛选过滤方法，避免冗余特征）

  • 过滤方法被广泛使用
  • 他们使用数据集的一般特征，并为特征分配权重，从这些特征中选择了许多得分最高的特征
  • 最常见的过滤方法是使用组级统计测试
    1. 首先使用不同的统计检验，如T检验、Welchs T检验和ranksum-test来识别具有组间差异的功能连接。
    2. 然后将这些功能连接作为分类方法的输入特征
  • 会出现的问题：
    • 可能导致分类结果的偏倚
    • 特征通常是基于它们的P值来选择的
  • 一种评分标准 Fisher评分
  • 其他的分类算法
    • 基于相关的特征选择(correlation-based-feature selection, CFS)
    • 基于RELIEF的算法
    • 上述两个算法特征选择采用了最小冗余、最大关联(mRMR)算法。，结果也很满意
  • 基于过滤的缺点
    • 过滤方法选择的特征没有经过优化以适应任何特定的分类器
    • 一些过滤方法由于忽略了特征之间的关系而倾向于选择冗余特征

3.1.2 Wrapper方法和 Embedded 方法

  • wrapper方法
    • 原理： 使用分类器或预测模型对特性进行排序。这类方法评估了不同特征组合的分类性能，并试图识别出能够提供最大分辨能力的特征的最优子集。
    • 递归特征消除(RFE)是最常用的一种方法，它通过递归地考虑越来越小的特征组合来选择特征
    • 遗传算法(GA)家族
    • 优势：
      • 倾向于选择性能比筛选器方法更好的特性
      • 可以为特定类型的分类器提供最佳特性选择
    • 缺点：
      • 如果观察值不大，wrapper方法可能会过度拟合
      • wrapper方法的计算成本要大得多，因为它们需要递归地创建分类器。
  • Embedded 方法
    • 区别于wrapper：学习部分和特征选择部分是不可分割的
    • 常用方法
      • LASSO方法
      • 岭回归
      • 弹性网络算法
    • 优势：因为Embedded方法选择特定于分类器的特性，所以它们更快，计算代价更小。

3.2 预测和分类的机器学习模型

3.2.1 传统机器学习模型

  • 支持向量机(SVM)是目前最流行的方法
    • 标准支持向量机是一种二元分类器，它推广了最佳分离超平面以更好地分离不同组的数据
    • 基本思想：找到与另一个类的观测值最接近的一个类的观测值
  • 线性判别分析（LDA）
    • 基本思想：它将特征投射到低维空间中，在低维空间中，不同组的数据可以最大限度地分离
    • LDA和PCA区别：成分分析用于寻找使数据方差最大的轴，而LDA用于寻找使多个组之间的距离最大的轴

3.2.2 深度学习模型

  • 优点：深度学习方法能够通过使用具有不同复杂程度的层次结构，直接从原始数据中学习最优表示，在较低层次上对原始输入数据空间具有更多的信息性。
  • 常用方法：
    • 人工神经网络(ANN)
      • ANN通过构建人工神经元和神经元之间的连接层来学习从例子中完成任务
    • 深度信念网络(Deep belief network, DBN)
      • DBN不是使用确定性函数和重构误差(如自动编码器)，而是使用最大似然估计进行预训练。
  • 面临问题：
    • 时间和计算资源的数量
    • 潜在的过度拟合问题
    • 从深度学习方法中获得的结果的可解释性

4.脑功能连接在分类脑疾病方面的应用

  • 精分和双相情感障碍
  • 自闭症谱系障碍和注意缺陷多动症
  • 阿尔兹海默症症和轻度认知障碍障碍

5. 在确定脑障碍的生物标记物和个体的分类任务上的挑战和困难

5.1 缺少诊断的“金标准”

  • 诊断仅由可观察到的症状决定，而访谈和病史是影响诊断的主要因素
  • 我们非常需要根据生物数据，并以客观和定量验证的方法确立领域标准

5.2 从高维测量中识别准确的病理特征作为生物标志物是很困难的

考虑神经影响数据通常有比样本数量更多的特性，因此，减少特征或者说使用特征工程选取更好的特征作为模型判别的依据可能是更好的。
